      This page billing database can be updated by uploading a update file. The format of the update file must be as below:

      The file must be of type txt

      Each line in the text file has 6 values separated by tabs or spaces

      Each line holds the bill data for one user in the following order

      aadhar_id bill_date bill_due_date bill_amount consumption(units) payment_status

      All the dates are in the format 'yyyy-mm-dd' or 'yyyy/mm/dd'

      If there is a discrepancy in the data, all the processed bills will be deleted and an error message will be displayed

      Make sure you logout before leaving to ensure that your account will not be misused
